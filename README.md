# automationpractice
This is demo project with automation testing
Using:
- Mocha Framework
- WebdriverIO Runner
- Allure report
Website: https://demo.nopcommerce.com/

# how to run
1. Get latest resource from git
2. Create folder 'error_shot' in root
3. Run 'npm install'
4. Run 'npm test wdio.config.js'

# how to run report
1. Run 'npm run report'
